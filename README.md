# 🎮 Educ-X — Learn. Play. Level Up.

## 📖 About
Educ-X is a command-line application built in Python that transforms the way 
students study. By bringing game mechanics into the learning experience, Educ-X 
makes revision sessions more engaging, more effective, and — dare we say it — 
actually fun.

Forget passive rereading. With Educ-X, students interact with their course 
material, earn progress, and keep coming back for more.

## 🎯 Who is it for?
Educ-X is designed for students at any level who want to :
- Retain their lessons more effectively
- Stay motivated during revision sessions
- Turn study time into a rewarding challenge

## ✨ Features
- 🕹️ Gamified learning experience through the command line
- 📚 Course content interaction and retention tools
- 🏆 Progress tracking to keep students motivated
- ⚡ Lightweight and fast — runs entirely in the terminal

## 🛠️ Installation

### Prerequisites
Make sure you have **Python 3.8 or higher** installed on your machine.
Check your version with :
```bash
python --version
```

### Steps

**1. Clone the repository**
```bash
git clone https://github.com/Orlande10/Educ-X-code.git
```

**2. Navigate into the project folder**
```bash
cd Educ-X-code
```

**3. Install dependencies** *(if any)*
```bash
pip install -r requirements.txt
```

**4. Run the application**
```bash
python main.py
```
